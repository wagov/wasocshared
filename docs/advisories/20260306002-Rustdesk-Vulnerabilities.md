# Rustdesk Vulnerabilities - 20260306002

## Overview

Based on recent vulnerability advisories published for RustDesk (including multiple CVEs disclosed in early 2026 affecting the client and Server Pro versions), the WASOC has identified potential indicators of RustDesk software present within your environment.

## What is vulnerable?

| Product(s) Affected | Version(s) | CVE                                                                      | CVSS| Severity     |
| ------------------- | ---------- | ------------------------------------------------------------------------ | --- | ------------ |
| RustDesk Client     | Version ≤ 1.4.5   | [CVE-2026-30789](https://nvd.nist.gov/vuln/detail/CVE-2026-30789) | 9.3 | **Critical** |
| RustDesk Client     | Version ≤ 1.4.5   | [CVE-2026-30790](https://nvd.nist.gov/vuln/detail/CVE-2026-30790) | 9.3 | **Critical** |
| RustDesk Client     | Version ≤ 1.4.5   | [CVE-2026-30792](https://nvd.nist.gov/vuln/detail/CVE-2026-30792) | 9.1 | **Critical** |
| RustDesk Client     | Version ≤ 1.4.5   | [CVE-2026-30793](https://nvd.nist.gov/vuln/detail/CVE-2026-30793) | 9.3 | **Critical** |
| RustDesk Client     | Version ≤ 1.4.5   | [CVE-2026-30794](https://nvd.nist.gov/vuln/detail/CVE-2026-30794) | 9.1 | **Critical** |
| RustDesk Client     | Version ≤ 1.4.5 (all Flutter-based builds)  | [CVE-2026-30797](https://nvd.nist.gov/vuln/detail/CVE-2026-30797) | 9.3 | **Critical** |


## What has been observed?

The WASOC has not received any reports of exploitation of this vulnerability on Western Australian Government networks at the time of writing.

## Recommendation

The WASOC recommends administrators apply the solutions as per vendor instructions to all affected devices within expected timeframes (refer [Patch Management](../guidelines/patch-management.md)):
